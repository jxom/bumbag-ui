# Select

```jsx
import { Select } from 'fannypack'
```

### Basic Usage

The `Select` component renders a `<select>` field with options.

```.jsx
<Select
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Default values

For uncontrolled select field, set a default value using the `defaultValue` prop.

```.jsx
<Select
  defaultValue="overcast"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Placeholders

Add a placeholder to a select field with the `placeholder` prop.

```.jsx
<Select
  placeholder="Select weather..."
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Disabled

Make the select field disabled with the `disabled` prop.

```.jsx
<Select
  disabled
  placeholder="Select weather..."
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Full width

To make the select field span full width, add the `isFullWidth` prop.

```.jsx
<Select
  isFullWidth
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Addons

You can use the [Group component](/components/group) to add on stuff to your `<Select>`.

```.jsx
<Group>
  <Select
    options={[
      { label: 'Yes', value: 'yes' },
      { label: 'No', value: 'no' }
    ]}
  />
  <Button palette="primary">Submit</Button>
</Group>
```

### Controlled Usage

The previous examples are examples of uncontrolled usage. To control the value yourself, use a combination of `onChange` and `value`.

Note: Do not use `defaultValue` with controlled inputs!

```.jsx
<Component initialState={{ value: 'windy' }}>
  {({ state, setState }) => (
    <Select
      onChange={e => setState({ value: e.target.value })}
      value={state.value}
      options={[
        { label: 'Sunny', value: 'sunny' },
        { label: 'Windy', value: 'windy' },
        { label: 'Overcast', value: 'overcast' }
      ]}
    />
  )}
</Component>
```

### Accessibility

Using an accessibility label (`a11yLabel`), will enable screen readers to read the contents of the label.

```.jsx
<Select
  a11yLabel="Weather"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

# SelectField

### Import

```jsx
import { SelectField } from 'fannypack'
```

### Basic Usage

The `<SelectField>` component contains of the [`<Select>` component](#select) wrapped by the [`<FieldWrapper>` component](/form/fieldwrapper).

It accepts a combination of [`<Select>` props](/form/select#select-props) and [`<FieldWrapper>` props](/form/fieldwrapper#props).

```.jsx
<SelectField
  a11yId="weather"
  label="Weather"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```
