# Spinners

### Import

```jsx
import { Spinner } from 'fannypack'
```

### Basic Usage

```.jsx
<Spinner />
```

### Sizes

```.jsx
<Spinner size="small" />
<Spinner />
<Spinner size="medium" />
<Spinner size="large" />
<Spinner size="50px" />
```

### Colors

```.jsx
<Spinner color="success" />
<Spinner color="danger" />
<Spinner color="warning" />
```

### Progress

You can indicate progress on your spinner by adding a `value` prop.

```.jsx
<Spinner value={25} />
<Spinner value={50} size="medium" />
<Spinner value={75} size="large" />
```
