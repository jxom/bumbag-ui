# HighlightedCode

The <code>&#60;HighlightedCode&#62;</code> component uses [Prism React Renderer](https://github.com/FormidableLabs/prism-react-rendere) under the hood.

## Install

```
yarn add @fannypack/addon-highlighted-code
// or npm
npm install @fannypack/addon-highlighted-code
```

## Import

```javascript
import HighlightedCode from '@fannypack/addon-highlighted-code'
```

## Usage

```jsx-live
<HighlightedCode code="console.log('Hello world')" language="javascript" />
<HighlightedCode
  isBlock
  language="jsx"
  marginTop="major-3"
  code={`<Box>
  <Text color="primary">Hello world</Text>
</Box>`}
/>
```
