# Code

## Import

```jsx
import { Code } from 'bumbag'
```

## Usage

```jsx-live
<Code>Hello world!</Code>
```

### Code blocks

A code block can be rendered by setting the `isBlock` prop. This will wrap the code in a `<pre>`.

```jsx-live
<Code isBlock>
{`function log(message) {
  console.log(message);
}`}
</Code>
```

### Colors

You can style code with colors from the palette.

```jsx-live
<Code palette="primary">Hello world</Code><br/>
<Code palette="secondary">Hello world</Code><br/>
<Code palette="success">Hello world</Code>
```
