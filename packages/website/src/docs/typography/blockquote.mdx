# Blockquote

## Import

```jsx
import { Blockquote } from 'bumbag'
```

## Usage

```jsx-live
<Blockquote>
  <Paragraph>
    "Out here, survival is the name of the game.
    Only it's not a game; it's deadly serious."
  </Paragraph>
  <Paragraph>- Russell Coight</Paragraph>
</Blockquote>
```

### Border styling

The left border can be modified using the primitive style props:

```jsx-live
<Blockquote borderColor="primary">
  <Paragraph>
    "Out here, survival is the name of the game.
    Only it's not a game; it's deadly serious."
  </Paragraph>
  <Paragraph>- Russell Coight</Paragraph>
</Blockquote>
```

Or you can hide the border:

```jsx-live
<Blockquote border="none">
  <Paragraph>
    "Out here, survival is the name of the game.
    Only it's not a game; it's deadly serious."
  </Paragraph>
  <Paragraph>- Russell Coight</Paragraph>
</Blockquote>
```
