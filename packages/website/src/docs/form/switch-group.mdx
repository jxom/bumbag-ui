# SwitchGroup

A `<SwitchGroup>` can be used to render a set of [Switch components](/form/switch).

## Import

```jsx
import { SwitchGroup } from 'bumbag'
```

## Usage

```jsx-live
<SwitchGroup
  name="weather"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Horizontal group

```jsx-live
<SwitchGroup
  orientation="horizontal"
  name="weather2"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Disabled

You can disable a radio group with the `disabled` prop. If you only want to disable one option, add the `disabled: true` attribute to the option.

```jsx-live
<SwitchGroup
  name="weather"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy', disabled: true },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

```jsx-live
<SwitchGroup
  disabled
  name="weather"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### States

A SwitchGroup can use different states (as per palette) such as `danger`, `success` and `warning`.

```jsx-live
<SwitchGroup
  state="danger"
  name="weather4"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Default value

Specify a default checked value with the `defaultValue` prop.

```jsx-live
<SwitchGroup
  defaultValue={['overcast', 'sunny']}
  name="weather5"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Fields

The `<SwitchGroupField>` component contains of the [`<SwitchGroup>` component](#SwitchGroup) wrapped by the [`<FieldWrapper>` component](/form/fieldwrapper).

It accepts a combination of [`<SwitchGroup>` props](/form/switch-group/#props) and [`<FieldWrapper>` props](/form/field-wrapper/#props).

```jsx-live
<SwitchGroupField
  label="What's the weather?"
  name="weather5"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```
