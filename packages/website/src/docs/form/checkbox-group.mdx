# CheckboxGroup

A `<CheckboxGroup>` can be used to render a set of [Checkbox components](/form/checkbox).

## Import

```jsx
import { CheckboxGroup } from 'bumbag'
```

## Usage

```jsx-live
<CheckboxGroup
  name="weather"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Horizontal group

```jsx-live
<CheckboxGroup
  orientation="horizontal"
  name="weather2"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Disabled

You can disable a radio group with the `disabled` prop. If you only want to disable one option, add the `disabled: true` attribute to the option.

```jsx-live
<CheckboxGroup
  name="weather"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy', disabled: true },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

```jsx-live
<CheckboxGroup
  disabled
  name="weather"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### States

A CheckboxGroup can use different states (as per palette) such as `danger`, `success` and `warning`.

```jsx-live
<CheckboxGroup
  state="danger"
  name="weather4"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Default value

Specify a default checked value with the `defaultValue` prop.

```jsx-live
<CheckboxGroup
  defaultValue={['overcast', 'sunny']}
  name="weather5"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Fields

The `<CheckboxGroupField>` component contains of the [`<CheckboxGroup>` component](#CheckboxGroup) wrapped by the [`<FieldWrapper>` component](/form/fieldwrapper).

It accepts a combination of [`<CheckboxGroup>` props](/form/checkbox-group/#props) and [`<FieldWrapper>` props](/form/field-wrapper/#props).

```jsx-live
<CheckboxGroupField
  label="What's the weather?"
  name="weather5"
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

## Accessibility

The `<CheckboxGroup>` component follows the [WAI ARIA Checkbox Pattern](https://www.w3.org/TR/wai-aria-practices/#checkbox).

### Rules

- The outer wrapper of the `CheckboxGroup` has a role of `group`.

#### Patterns

- The checkbox has a role of `checkbox`.
- When the checkbox has focus, pressing the <Key>Space</Key> key changes the state of the checkbox.

#### References

- [WAI ARIA Checkbox Pattern](https://www.w3.org/TR/wai-aria-practices/#checkbox)
