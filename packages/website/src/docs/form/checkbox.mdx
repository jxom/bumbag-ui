# Checkbox

Checkboxes are used when a user needs to select one or more values from a series of options, or they can be used as a binary toggle.

## Import

```jsx
import { Checkbox, CheckboxField } from 'bumbag'
```

## Usage

```jsx-live
<Checkbox label="Check me" />
```

### Fields

You can also use the `<CheckboxField>` component, which internally has a [FieldWrapper](/form/field-wrapper) to wrap form controls such as labels, hint text and validation text.

It accepts a combination of [Checkbox props](/form/checkbox/#props) and [FieldWrapper props](/form/field-wrapper/#props).

```jsx-live
<CheckboxField
  checkboxLabel="Check me"
  label="Check below to subscribe"
/>
```

### Disabled

Make the checkbox disabled with the `disabled` prop.

```jsx-live
<Checkbox disabled label="Click me" />
<Checkbox checked disabled label="Click me" />
```

### Indeterminate

```jsx-live
<Checkbox label="Click me" checked indeterminate />
```

### States

A checkbox can use different states (as per palette) such as `danger`, `success` and `warning`.

```jsx-live
<Checkbox state="danger" label="Click me" />
<Checkbox state="success" label="Click me" />
<Checkbox state="warning" label="Click me" />
<Checkbox state="primary" label="Click me" />
```

### Controlled Usage

The previous examples are examples of uncontrolled usage. To control the value yourself, use a combination of `onChange` and `value`.

Note: Do not use `defaultValue` with controlled inputs!

```function-live
function Example() {
  const [checked, setChecked] = React.useState(true);

  return (
    <Checkbox
      label="Click me"
      checked={checked}
      onChange={e => setChecked(e.target.checked)}
    />
  );
}
```

## Accessibility

The `<Checkbox>` component follows the [WAI ARIA Checkbox Pattern](https://www.w3.org/TR/wai-aria-practices/#checkbox).

### Rules

- The checkbox must have a `label` prop to identify it's label. If a `label` prop is not provided, an `aria-label` or `aria-labelledby` prop can be specified.

#### Patterns

- The checkbox has a role of `checkbox`.
- The checkbox has an accessible label specified via the `label`, `aria-labelledby` or `aria-label` prop.
- When the checkbox has focus, pressing the <Key>Space</Key> key changes the state of the checkbox.

#### References

- [WAI ARIA Checkbox Pattern](https://www.w3.org/TR/wai-aria-practices/#checkbox)
