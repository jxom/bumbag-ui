# Stack

## Import

```jsx
import { Stack } from 'fannypack';
```

## Usage

The `<Stack>` component that assists in creating layouts where it's children are evenly spaced.

```jsx-live
<Stack>
  <Card>First</Card>
  <Card>
    <Stack>
      <Card variant="bordered">Second</Card>
      <Card variant="bordered">Third</Card>
    </Stack>
  </Card>
  <Stack orientation="horizontal">
    <Card>Fourth</Card>
    <Card>Fifth</Card>
  </Stack>
</Stack>
```

### Gap

You can specify a gap spacing on your `<Stack>` with the `spacing` prop.

```jsx-live
<Stack spacing="major-2">
  <Card>First</Card>
  <Card>Second</Card>
  <Card>Third</Card>
</Stack>
```

### Vertical breakpoints

You can specify a gap spacing on your `<Stack>` with the `spacing` prop.

```jsx-live
<Stack spacing="major-2">
  <Card>First</Card>
  <Card>Second</Card>
  <Card>Third</Card>
</Stack>
```
