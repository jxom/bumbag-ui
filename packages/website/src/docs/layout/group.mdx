# Group

## Import

```jsx
import { Group } from 'bumbag';
```

## Usage

The `<Group>` component groups components together with their respective `border-radius`.

```jsx-live
<Group>
  <Box backgroundColor="primary" color="white" padding="0.5rem 1rem">
    Hello
  </Box>
  <Button>Send</Button>
  <Button>Cancel</Button>
</Group>
```

### Vertical groups

A group can also be aligned vertically.

```jsx-live
<Group orientation="vertical">
  <Box backgroundColor="primary" color="white" padding="0.5rem 1rem">
    Hello
  </Box>
  <Button>Send</Button>
  <Button>Cancel</Button>
</Group>
```

### Responsive breakpoint

A group can snap from horizontal to vertical after a given breakpoint with the `verticalBelow` prop.

```jsx-live
<Group verticalBelow="widescreen">
  <Box backgroundColor="primary" color="white" padding="0.5rem 1rem">
    Hello
  </Box>
  <Button>Send</Button>
  <Button>Cancel</Button>
</Group>
```
