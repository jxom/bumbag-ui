# Level

## Import

```jsx
import { Level } from 'bumbag';
```

## Usage

The `<Level>` component that assists in creating layouts where it's children are evenly spaced.

```jsx-live
<Level>
  <Box backgroundColor="primary" width="50px" height="50px" />
  <Box backgroundColor="primary" width="50px" height="50px" />
</Level>
```

### Vertical breakpoint

By default, `Level` will turn vertical under the `desktop` breakpoint. However, you can change this with the `verticalBelow` prop.

```jsx-live
<Level verticalBelow="widescreen">
  <Box backgroundColor="primary" width="50px" height="50px" />
  <Box backgroundColor="primary" width="50px" height="50px" />
</Level>
```

You can disable the vertical snapping by setting `verticalBelow` to `null`.

```jsx-live
<Level verticalBelow={null}>
  <Box backgroundColor="primary" width="50px" height="50px" />
  <Box backgroundColor="primary" width="50px" height="50px" />
</Level>
```

### Vertical spacing

You can modify the space between the vertical items with the `spacing` prop.

```jsx-live
<Level spacing="major-4">
  <Box backgroundColor="primary" width="50px" height="50px" />
  <Box backgroundColor="primary" width="50px" height="50px" />
</Level>
```

### Vertical alignment

You can modify the alignment of the vertical items with the `alignX` prop.

```jsx-live
<Level alignX="right">
  <Box backgroundColor="primary" width="50px" height="50px" />
  <Box backgroundColor="primary" width="50px" height="50px" />
</Level>
```
