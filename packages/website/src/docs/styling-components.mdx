# Styling components

Bumbag comes with an `applyTheme` function which allows you to create components that have styles attached to them.

## Creating a basic styled component

To create a styled component, import the `applyTheme` function and then invoke it with your desired component & template literal string containing your CSS:

```function-live noInline
const FancyButton = applyTheme(Button, {
  styles: {
    base: {
      color: 'primary'
    }
  },
  defaultProps: {
    variant: 'ghost'
  }
});

function Example() {
  return (
    <FancyButton>Hello world</FancyButton>
  );
}

render(Example)
```

## Generating styles based from props

You can generate styles based on your props (and changes in props) by supplying a function (`props => ...`) to your theme attribute:

```function-live noInline
const FancyBox = applyTheme(Box, {
  styles: {
    base: props => ({
      backgroundColor: props.palette,
      ...(props.type === 'rounded' && ({ borderRadius: '4' }))
    })
  }
});

function Example() {
  return (
    <FancyBox palette="primary" color="white" type="rounded">Hello world</FancyBox>
  );
}

render(Example)
```


## Generating styles based from the theme

You can generate styles based off values in your [Bumbag global theme](/theming) by using the `theme` variable stored in the `props` object:

```function-live noInline
const FancyBox = applyTheme(Box, {
  styles: {
    base: props => ({
      backgroundColor: props.theme.palette.primary,
      color: 'white'
    })
  }
});

function Example() {
  return (
    <FancyBox>Hello world</FancyBox>
  );
}

render(Example)
```
