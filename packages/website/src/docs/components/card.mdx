# Card

## Import

```jsx
import { Card } from 'bumbag'
```

## Usage

```jsx-live
<Card>Hello world</Card>
<Card variant="shadowed">Hello world</Card>
<Card variant="bordered">Hello world</Card>
```

### Header

You can render a header for the card using the `title` prop.

```jsx-live
<Card title="This is a title">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
  eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
  augue sit amet justo.
</Card>
```

### Header with addons

You can render a header with addons for the card using the `headerAddon` prop.

```jsx-live
<Card
  title="This is a title"
  headerAddon={
    <Set>
      <Button palette="primary" variant="link">Action 1</Button>
      <Button palette="primary" variant="link">Action 2</Button>
    </Set>
  }
>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
  eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
  augue sit amet justo.
</Card>
```

### Footer

You can render a footer for the card using the `footer` prop.

```jsx-live
<Card
  title="This is a title"
  footer={
    <ActionButtons
      justifyContent="flex-end"
      width="100%"
    />
  }
>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
  eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
  augue sit amet justo.
</Card>
```

### Standalone

```jsx-live
<Card standalone>
  <Card.Header>
    <Card.Title>This is a title</Card.Title>
  </Card.Header>
  <Card.Content>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
    eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
    augue sit amet justo.
  </Card.Content>
  <Card.Footer>
    <Button>Submit</Button>
  </Card.Footer>
</Card>
```

## Accessibility

### Patterns

- If a card has a **title**, then the `aria-labelledby` attribute is set on the component.
- If a card has a **description**, then the `aria-describedby` attribute is set on the component.
