# Breadcrumb

A breadcrumb consists of a list of links from the parent page to the current page.

## Import

```jsx
import { Breadcrumb } from 'bumbag';
```

## Usage

```jsx-live
<Breadcrumb>
  <Breadcrumb.Item>
    <Breadcrumb.Link href="#">Home</Breadcrumb.Link>
  </Breadcrumb.Item>
  <Breadcrumb.Item>
    <Breadcrumb.Link href="#">Business</Breadcrumb.Link>
  </Breadcrumb.Item>
  <Breadcrumb.Item isCurrent>
    <Breadcrumb.Link href="#">Staff Members</Breadcrumb.Link>
  </Breadcrumb.Item>
</Breadcrumb>
```

### Custom separators

You can render a custom separator for your breadcrumb with the `separator` prop.

```jsx-live
<Breadcrumb separator="-">
  <Breadcrumb.Item>
    <Breadcrumb.Link href="#">Home</Breadcrumb.Link>
  </Breadcrumb.Item>
  <Breadcrumb.Item>
    <Breadcrumb.Link href="#">Business</Breadcrumb.Link>
  </Breadcrumb.Item>
  <Breadcrumb.Item isCurrent>
    <Breadcrumb.Link href="#">Staff Members</Breadcrumb.Link>
  </Breadcrumb.Item>
</Breadcrumb>
```

You can even use components as a separator.

```jsx-live
<Breadcrumb separator={<Icon icon="solid-chevron-right" color="gray100" fontSize="150" />}>
  <Breadcrumb.Item>
    <Breadcrumb.Link href="#">Home</Breadcrumb.Link>
  </Breadcrumb.Item>
  <Breadcrumb.Item>
    <Breadcrumb.Link href="#">Business</Breadcrumb.Link>
  </Breadcrumb.Item>
  <Breadcrumb.Item isCurrent>
    <Breadcrumb.Link href="#">Staff Members</Breadcrumb.Link>
  </Breadcrumb.Item>
</Breadcrumb>
```

You can specify the `separator` on a `<Breadcrumb.Item>` level.

```jsx-live
<Breadcrumb>
  <Breadcrumb.Item separator={<Icon icon="solid-chevron-right" color="gray100" fontSize="150" />}>
    <Breadcrumb.Link href="#">Home</Breadcrumb.Link>
  </Breadcrumb.Item>
  <Breadcrumb.Item>
    <Breadcrumb.Link href="#">Business</Breadcrumb.Link>
  </Breadcrumb.Item>
  <Breadcrumb.Item isCurrent>
    <Breadcrumb.Link href="#">Staff Members</Breadcrumb.Link>
  </Breadcrumb.Item>
</Breadcrumb>
```

You can use the `<Breadcrumb.Separator>` component directly.

```jsx-live
<Breadcrumb hasSeparator={false}>
  <Breadcrumb.Item>
    <Breadcrumb.Link href="#">Home</Breadcrumb.Link>
    <Breadcrumb.Separator />
  </Breadcrumb.Item>
  <Breadcrumb.Item>
    <Breadcrumb.Link href="#">Business</Breadcrumb.Link>
    <Breadcrumb.Separator>
      <Icon icon="solid-chevron-right" color="gray100" fontSize="150" />
    </Breadcrumb.Separator>
  </Breadcrumb.Item>
  <Breadcrumb.Item isCurrent>
    <Breadcrumb.Link href="#">Staff Members</Breadcrumb.Link>
  </Breadcrumb.Item>
</Breadcrumb>
```

## Accessibility

The `<Breadcrumb>` component follows the [WAI ARIA Breadcrumb Pattern](https://www.w3.org/TR/wai-aria-practices/#breadcrumb).

#### Patterns

- The `<Breadcrumb>` component renders as a `<nav>` component labelled as `"Breadcrumb"` by default.
- To prevent the separators being read by a screen reader, the role of `"presentation"` is applied to them.
- The link to the current page has `aria-current` set to page.

#### References

- [WAI ARIA Breadcrumb Pattern](https://www.w3.org/TR/wai-aria-practices/#breadcrumb)
- [WAI ARIA Breadcrumb Example](https://www.w3.org/TR/wai-aria-practices-1.1/examples/breadcrumb/index.html)
