# Popover

The `<Popover>` component is a lightweight dialog that floats beside a disclosure (a button).

## Import

```jsx
import { Popover } from 'bumbag';
```

## Usage

The `<Popover.Disclosure>` component will act as the trigger of your popover.

To automatically handle the state of your popover, wrap the `<Popover>` & `<Popover.Disclosure>` components in a `<Popover.State>`.

```jsx-live
<Popover.State>
  <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
  <Popover hasArrow>
    What was the best thing that happened to you today?
  </Popover>
</Popover.State>
```

> If you wish to control the state of your popover, see [controlled usage](#controlled-usage).

### Placement

You can control the placement of your popover by using the `placement` prop.

```jsx-live
<Popover.State placement="top">
  <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
  <Popover hasArrow>
    What was the best thing that happened to you today?
  </Popover>
</Popover.State>
```

### Title

Render a title using the `title` prop.

```jsx-live
<Popover.State>
  <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
  <Popover title="Attention!">
    What was the best thing that happened to you today?
  </Popover>
</Popover.State>
```

### Footer

Render a footer using the `title` prop.

```jsx-live
<Popover.State>
  <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
  <Popover footer="This is a footer">
    What was the best thing that happened to you today?
  </Popover>
</Popover.State>
```

### Action buttons

```jsx-live
<Popover.State>
  <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
  <Popover showActionButtons actionButtonsProps={{ submitText: 'Save' }}>
    What was the best thing that happened to you today?
  </Popover>
</Popover.State>
```

### Close within popover

The `<Popover.State>` component also renders children as a function (render props).

You can retrieve utilities from the `popover` variable such as `hide` to hide the popover.

```jsx-live
<Popover.State>
  {popover => (
    <React.Fragment>
      <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
      <Popover hasArrow>
        What was the best thing that happened to you today?
        <Button onClick={popover.hide}>
          Close
        </Button>
      </Popover>
    </React.Fragment>
  )}
</Popover.State>
```

Or you can display a close button:

```jsx-live
<Popover.State>
  <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
  <Popover title="Attention!" showCloseButton>
    What was the best thing that happened to you today?
  </Popover>
</Popover.State>
```

### Render within a portal

You can render the popover within a [React Portal](https://reactjs.org/docs/portals.html).

```jsx-live
<Popover.State>
  <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
  <Popover usePortal>
    What was the best thing that happened to you today?
  </Popover>
</Popover.State>
```

### Setting initial focus

You can specify an initial element to put focus on using the `unstable_initialFocusRef` prop.

```function-live
function Example() {
  const buttonRef = React.useRef();

  return (
    <Popover.State>
      <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
      <Popover unstable_initialFocusRef={buttonRef}>
        What was the best thing that happened to you today?
        <Button>Button 1</Button>
        <Button ref={buttonRef}>Button 2</Button>
      </Popover>
    </Popover.State>
  )
}
```

### Setting final focus

You can specify an element focus on when the popover closes using the `unstable_finalFocusRef` prop.

```function-live
function Example() {
  const buttonRef = React.useRef();

  return (
    <Popover.State>
      <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
      <Button ref={buttonRef}>Button 2</Button>
      <Popover unstable_finalFocusRef={buttonRef}>
        What was the best thing that happened to you today?
      </Popover>
    </Popover.State>
  )
}
```

### Restricting closure

You can retrict closure (via click & esc) by using the `hideOnEsc` and `hideOnClickOutside` props.

```jsx-live
<Popover.State>
  {popover => (
    <React.Fragment>
      <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
      <Popover hideOnEsc={false} hideOnClickOutside={false}>
        What was the best thing that happened to you today?
        <Button onClick={popover.hide}>
          Close
        </Button>
      </Popover>
    </React.Fragment>
  )}
</Popover.State>
```

### Animation

To enable animation on your popover. You will have to set the `animated` flag on `<Popover.State>`

#### Fade

```jsx-live
<Popover.State animated>
  <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
  <Popover fade>
    What was the best thing that happened to you today?
  </Popover>
</Popover.State>
```

#### Slide

```jsx-live
<Popover.State animated>
  <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
  <Popover slide fade debug>
    What was the best thing that happened to you today?
  </Popover>
</Popover.State>
```

#### Expand

```jsx-live
<Popover.State animated>
  <Popover.Disclosure use={Button}>Open Popover</Popover.Disclosure>
  <Popover expand slide fade>
    What was the best thing that happened to you today?
  </Popover>
</Popover.State>
```

### Controlled usage

If you would like more control over the state of your popover, you can use either the `useState` hook, or the `<Popover.State>` render props.

```live
###tab=Hook,type=function-live
function Example() {
  const popover = Popover.useState();

  return (
    <React.Fragment>
      <Popover.Disclosure use={Button} {...popover}>Open Popover</Popover.Disclosure>
      <Popover hasArrow {...popover}>
        What was the best thing that happened to you today?
      </Popover>
    </React.Fragment>
  )
}
###

###tab=Render Props,type=jsx-live
<Popover.State>
  {popover => (
    <React.Fragment>
      <Popover.Disclosure use={Button} {...popover}>Open Popover</Popover.Disclosure>
      <Popover hasArrow {...popover}>
        What was the best thing that happened to you today?
      </Popover>
    </React.Fragment>
  )}
</Popover.State>
###
```

### Accessing internal state

Any descendant component of `Popover.State` can utilise `Popover.useContext` to access the [internal popover state](#popoverstate-return-values):

```jsx
function Example() {
  const { popover } = Popover.useContext();

  return (
    <Button onClick={popover.hide}>Hide popover</Button>
  )
}
```

## Accessibility

- `Popover` extends the accessibility features of the [Modal component](/utilities/modal).
