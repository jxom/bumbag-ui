# Badge

A badge can be commonly used as a notification or number indicator.

## Import

```jsx
import { Badge } from 'bumbag';
```

## Usage

```jsx-live
<Badge>
  1
</Badge>
```

### No content

```jsx-live
<Badge />
```

### Colors

```jsx-live
<Set spacing="minor-1">
  <Badge palette="primary">12</Badge>
  <Badge palette="secondary">12</Badge>
  <Badge palette="info">12</Badge>
  <Badge palette="success">12</Badge>
  <Badge palette="warning">12</Badge>
  <Badge palette="danger">12</Badge>
</Set>
```

### Sizes

```jsx-live
<Set spacing="minor-1">
  <Badge>
    1
  </Badge>
  <Badge size="medium">
    1
  </Badge>
  <Badge size="large">
    1
  </Badge>
</Set>
```

### Attachment

You can attach a badge onto a component by providing it as a child of the component.

```jsx-live
<Button>
  Hello world
  <Badge isAttached palette="danger" />
</Button>
<Button size="large" marginLeft="major-2">
  Hello world
  <Badge isAttached palette="danger" size="large">1</Badge>
</Button>
<Box backgroundColor="primary" color="white" marginTop="major-2">
  Hello world
  <Badge isAttached palette="danger">1</Badge>
</Box>
```
