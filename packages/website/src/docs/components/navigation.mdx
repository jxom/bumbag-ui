# Navigation

The `<Navigation>` component renders `<nav>` under the hood.

## Import

```jsx
import { Navigation } from 'bumbag'
```

## Usage

```jsx-live
<Navigation aria-label="External links">
  <List>
    <List.Item>
      <Link href="https://google.com">Google</Link>
    </List.Item>
    <List.Item>
      <Link href="https://github.com">GitHub</Link>
    </List.Item>
  </List>
</Navigation>
```

## Accessibility

The `<Navigation>` component follows the [WAI ARIA Navigation Pattern](https://www.w3.org/TR/wai-aria-practices/#aria_lh_navigation).

#### Patterns

- The `nav` element is used.
- If the `use` prop is set to anything other than `nav`, then a role of `navigation` will be set as an attribute.

#### References

- [WAI ARIA Navigation Pattern](https://www.w3.org/TR/wai-aria-practices/#aria_lh_navigation)
