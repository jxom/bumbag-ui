# Alignment

```jsx-live
<Box alignY="center" border="default" height="250px">
  I am centered vertically
</Box>
```

```jsx-live
<Box alignX="center" border="default" height="250px">
  I am centered horizontally
</Box>
```

```jsx-live
<Box alignX="center" alignY="center" border="default" height="250px">
  I am centered horizontally and vertically
</Box>
```

```jsx-live

  I am centered vertically
</Flex>
```

```jsx-live

```

