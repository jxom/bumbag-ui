# Clickable

The `<Clickable>` component is a utility component that implements all the interactions an interactive element needs to be fully accessible when it's not rendered as its respective native element. It is an export of [Reakit's Clickable](https://reakit.io/docs/clickable/)

## Import

```js
import { Clickable } from 'bumbag';
```

## Usage

```jsx-live
<Set>
  <Clickable
    use={Button}
    onClick={() => alert('Clicked')}
  >
    Clickable
  </Clickable>
  <Clickable
    use={Button}
    disabled
    onClick={() => alert('Clicked')}
  >
    Clickable
  </Clickable>
  <Clickable
    use={Button}
    disabled
    focusable
    onClick={() => alert('Clicked')}
  >
    Clickable
  </Clickable>
</Set>
```

## Accessibility

#### Patterns

- Pressing the <Key>Enter</Key> or <Key>Space</Key> keys triggers the button.
- `Clickable` extends the accessibility features of [Tabbable](/tabbable).
