# Picker

## Install

```
yarn add @bumbag-native/picker
// or npm
npm install @bumbag-native/picker
```

## Import

```js
import { Picker } from '@bumbag-native/picker'
```

## Usage

```jsx-live
<Picker
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }
  ]}
/>
```

### Dividers

```jsx-live
<Picker
  hasDividers
  options={[
    { label: 'Sunny', value: 'sunny' },
    { label: 'Windy', value: 'windy' },
    { label: 'Overcast', value: 'overcast' }

```
