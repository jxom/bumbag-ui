# Hidden

## Import

```jsx
import { Hidden } from 'fannypack';
```

## Usage

```function-live
function Example() {
  const hidden = Hidden.useState({ visible: true })

  return (
    <div>
      <Hidden.Disclosure {...hidden}>Toggle</Hidden.Disclosure>
      <Hidden {...hidden}>Hello world</Hidden>
    </div>
  )
}
```

### Composition

#### With `use`

```function-live
function Example() {
  const hidden = Hidden.useState({ visible: true })

  return (
    <div>
      <Hidden.Disclosure use={Button} {...hidden}>Toggle</Hidden.Disclosure>
      <Hidden use={Blockquote} borderColor="primary" backgroundColor="primaryTint" {...hidden}>Hello world</Hidden>
    </div>
  )
}
```

#### With Hooks

```function-live
function Example() {
  const hidden = Hidden.useState({ visible: true })
  const hiddenDisclosureProps = Hidden.Disclosure.useProps({ ...hidden });
  const hiddenProps = Hidden.useProps({ ...hidden });

  return (
    <div>
      <Button {...hiddenDisclosureProps}>Toggle</Button>
      <Blockquote borderColor="primary" backgroundColor="primaryTint" {...hiddenProps}>Hello world</Blockquote>
    </div>
  )
}
```
