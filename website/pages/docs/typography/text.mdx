# Text

The `<Text>` component renders a `<span>` with basic reset styles by default.

A text comes in two different forms:

- `Text.Block`: Text rendered as block display.
- `Text`: Text rendered as inline.

The inline `Text` component should only be rendered within a:

  - **Text.Block**
  - [Paragraph](/typography/paragraph)
  - [Heading](/typography/heading)
  - [Link.Block](/typography/link)
  - [List.Item](/typography/list)

## Import

```jsx
import { Text } from 'bumbag'
```

## Usage

```jsx-live
<Text.Block>
  Hello world! This is a bit of
  <Text fontWeight="bold">text</Text>.
</Text.Block>
```

### Other elements

```jsx-live
<Text use="strong">Bold</Text><br />
<Text use="i">Italic</Text><br />
<Text use="u">Underline</Text><br />
<Text use="abbr">I18N</Text><br />
```
