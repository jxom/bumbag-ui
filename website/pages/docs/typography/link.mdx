# Link

A link comes in two different forms:

- `Link.Block`: Link rendered as block display.
- `Link`: Link rendered as inline.

The inline `Link` component should only be rendered within a:

  - **Link.Block**
  - [Text.Block](/typography/text)
  - [Paragraph](/typography/paragraph)
  - [Heading](/typography/heading)
  - [List.Item](/typography/list)

## Import

```jsx
import { Link } from 'bumbag'
```

## Usage

```jsx-live
<Link.Block href="https://google.com">Click me</Link.Block>
<Text.Block>
  This is a text block with a
  <Link href="https://google.com">link</Link>.
</Text.Block>
```
