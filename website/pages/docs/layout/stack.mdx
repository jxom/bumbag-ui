# Stack

## Import

```jsx
import { Stack } from 'bumbag';
```

## Usage

The `<Stack>` component that assists in creating layouts where it's children are evenly spaced.

```jsx-live
<Stack>
  <Card>First</Card>
  <Card>
    <Stack>
      <Card variant="bordered">Second</Card>
      <Card variant="bordered">Third</Card>
    </Stack>
  </Card>
  <Stack orientation="horizontal">
    <Card>Fourth</Card>
    <Card>Fifth</Card>
  </Stack>
</Stack>
```

### Gap

You can specify a gap spacing on your `<Stack>` with the `spacing` prop.

```jsx-live
<Stack spacing="major-2">
  <Card>First</Card>
  <Card>Second</Card>
  <Card>Third</Card>
</Stack>
```

### Vertical breakpoints

You can set a `verticalBelow` breakpoint, in which the horizontal stack will switch to a vertical orientation under the specified breakpoint.

**Try drag the viewport with the slider**

```jsx-live-iframe
<Stack orientation="horizontal" verticalBelow="mobile">
  <Card>First</Card>
  <Card>Second</Card>
  <Card>Third</Card>
</Stack>
```

### Alignments

#### Horizontal alignment

```jsx-live
<Stack alignX="left">
  <Card>First</Card>
  <Card>Second</Card>
  <Card>Third</Card>
</Stack>
<Stack alignX="center">
  <Card>First</Card>
  <Card>Second</Card>
  <Card>Third</Card>
</Stack>
<Stack alignX="right">
  <Card>First</Card>
  <Card>Second</Card>
  <Card>Third</Card>
</Stack>
```

#### Vertical alignment

```jsx-live
<Stack alignY="top" height="300px" backgroundColor="white700">
  <Card>First</Card>
  <Card>Second</Card>
</Stack>
<Stack alignY="center" height="300px" backgroundColor="primaryTint">
  <Card>First</Card>
  <Card>Second</Card>
</Stack>
<Stack alignY="bottom" height="300px" backgroundColor="white700">
  <Card>First</Card>
  <Card>Second</Card>
</Stack>
```
