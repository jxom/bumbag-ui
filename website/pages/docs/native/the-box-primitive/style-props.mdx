# Style props

You're able to apply any CSS attribute as a prop to `<Box>`. These attributes can be passed through as camel-cased props. style props can be very useful when you just want to apply an adhoc style without pulling it out into it's own [themeable component](/native/styling-components).

```jsx-live
<Box
  backgroundColor="blue"
  height="50px"
  width="50px"
/>
```

## Colors

You can specify a [color from the palette](/native/palette) with any color-based CSS attribute.

[Read more...](/native/palette)

```jsx-live
<Box backgroundColor="primary" color="primaryTint">Hello world</Box>
```

### Reference

**Theme reference:** [`theme.palette`](/native/palette)

**Style props:** `color`, `backgroundColor`, `borderBlockEndColor`, `borderBlockStartColor`, `borderBottomColor`, `borderColor`, `borderInlineEndColor`, `borderInlineStartColor`, `borderLeftColor`, `borderRightColor`, `borderTopColor`, `borderBottomColor`, `caretColor`, `columnRuleColor`, `outlineColor`, `textDecorationColor`, `textEmphasisColor`

## Spacing

You can specify a [spacing from the theme](/native/spacing) for margins & paddings.

[Read more...](/native/spacing)

```jsx-live
<Box
  padding="major-2"
  backgroundColor="primary"
  color="white"
>
  Hello world
</Box>
```

There are also `paddingX`, `paddingY`, `marginX` and `marginY` props to specify padding or margin in a particular direction.

```jsx-live
<Box
  paddingY="major-2"
  paddingX="major-4"
  backgroundColor="primary"
  color="white"
>
  Hello world
</Box>
```

### Reference

**Theme reference:** [`theme.spacing`](/native/spacing/#theming)

**Style props:** `margin`, `marginLeft`, `marginRight`, `marginTop`, `marginBottom`, `marginX`, `marginY`, `padding`, `paddingLeft`, `paddingRight`, `paddingTop`, `paddingBottom`, `paddingX`, `paddingY`, `top`, `left`, `bottom`, `right`, `height`, `width`, `minWidth`, `minHeight`, `maxWidth`, `maxHeight`

## Fonts & typography

You can specify a [font size from the theme](/native/fonts) for font sizes.

```jsx-live
<Text fontSize="400">Hello world</Text>
```

You can specify a [font weight from the theme](/native/fonts) for font weights.

```jsx-live
<Text fontWeight="semibold">Hello world</Text>
```

You can specify a [font family from the theme](/native/fonts) for font families.

```jsx-live
<Text fontFamily="Comic Sans MS">Hello world</Text>
```

You can specify a [letter spacing from the theme](/native/fonts) for letter spacings.

```jsx-live
<Text letterSpacing="100">Hello world</Text>
```

You can specify a [line height from the theme](/native/fonts) for line heights.

```jsx-live
<Text lineHeight="100">Hello<br/>world</Text>
```

[Read more...](/native/fonts)
