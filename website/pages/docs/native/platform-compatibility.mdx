# Platform compatibility

Please note the following:

- The components in the `bumbag` package are only compatible on the web,
- The web also supports `bumbag-native` components,
- React Native only supports components in `bumbag-native`.


| Package         | Web (React) | React Native |
| --------------- | ----------- | ------------ |
| `bumbag`        | ✅          | ❌           |
| `bumbag-native` | ✅          | ✅           |
