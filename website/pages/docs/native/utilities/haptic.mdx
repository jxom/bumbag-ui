# Haptic

## Install

> This component requires you to install `@bumbag-native/haptic` as a separate dependency.

## Import

```js
import { Haptic } from "@bumbag-native/haptic";
```

## Usage

```jsx-live
<Haptic>
  <Button>
    Continue
  </Button>
</Haptic>
```
