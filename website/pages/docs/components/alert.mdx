# Alerts

The `<Alert>` component displays a **brief** & **important** message in a way that attracts a user's attention without interrupting a user's task.

If you are needing to display more descriptive, but important message to a user, consider using the [`<Callout>` component](/components/callout).

## Import

```jsx
import { Alert } from 'bumbag'
```

## Usage

```jsx-live
<Alert
  title="Did you know?"
  showCloseButton
  onClickClose={() => alert('Closed!')}
>
  Bumbag is a React UI Kit.
</Alert>
```

### Titles

You can set a title with the `title` prop.

```jsx-live
<Alert title="Did you know?">
  Bumbag is a React UI Kit.
</Alert>
<Alert title="Did you know?" isInline>
  Bumbag is a React UI Kit.
</Alert>
```

### Types

An alert can take a type of `"info"`, `"success"`, `"warning"` or `"danger"`.

```jsx-live
<Alert title="Did you know?" type="info">
  Bumbag is a React UI Kit.
</Alert>
<Alert title="Action applied" type="success">
  Your action was applied successfully.
</Alert>
<Alert title="Be careful" type="warning">
  Performing this action can be harmful.
</Alert>
<Alert title="An error occurred" type="danger">
  Unable to apply your action.
</Alert>
```

### Accents

Alerts can have bordered accents.

```jsx-live
<Alert accent>
  Bumbag is a React UI Kit.
</Alert>
<Alert accent="top">
  Bumbag is a React UI Kit.
</Alert>
<Alert accent="bottom">
  Bumbag is a React UI Kit.
</Alert>
```

### Variants

The `<Alert>` component comes with **three** default variants:

- An alert with a **border**,
- An alert with a **tint**,
- A **filled** alert.

You can even [make your own Alert variant](/variants).

#### Bordered

```jsx-live
<Alert variant="bordered" title="Did you know?" type="info">
  Bumbag is a React UI Kit.
</Alert>
<Alert variant="bordered" title="Action applied" type="success">
  Your action was applied successfully.
</Alert>
<Alert variant="bordered" title="Be careful" type="warning">
  Performing this action can be harmful.
</Alert>
<Alert variant="bordered" title="An error occurred" type="danger">
```

## Accessibility

The `<Alert>` component follows the [WAI ARIA Alert Pattern](https://www.w3.org/TR/wai-aria-practices/#alert).

#### Patterns

- The component has a role of `"alert"`.
- If an alert has a **title**, then the `aria-labelledby` attribute is set on the component.
- If an alert has a **description**, then the `aria-describedby` attribute is set on the component.

#### References

- [WAI ARIA Alert Pattern](https://www.w3.org/TR/wai-aria-practices/#alert)
