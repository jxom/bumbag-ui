# PageWithSidebar

## Import

```jsx
import { Page } from 'bumbag';
```

## Usage

The `<PageWithSidebar>` component enables you to build a page layout with a sidebar.

Provide your sidebar component to the `sidebar` prop, and the rest of your page as children.

```jsx
<PageWithSidebar sidebar={<Box>This is a sidebar</Box>}>
  Hello world
</PageWithSidebar>
```

[See it in playroom](https://bumbag.style/playroom/#?code=N4Igxg9gJgpiBcIA8AFAhgcxgdQJYBcALAZV1gCM0AnAAgGcyZKqBeYJAIQgA8A%2BAFUK46NYTTT1GzJAHoufAL68AOgDsaNABIwANjog0A7hCo6oa2eix4ipCtV4gFQA)

### Sidebar placement

You can control which side of the screen the sidebar is on with the `sidebarPlacement` prop.

```jsx
<PageWithSidebar
  sidebar={<Box>This is a sidebar</Box>}
  sidebarPlacement="right"
>
  Hello world
</PageWithSidebar>
```

### Toggling the sidebar

You can toggle the sidebar with your own component using the `<PageWithSidebar.Disclosure>` component.

```jsx-live
<PageWithSidebar.Disclosure use={Button}>
  Toggle sidebar
</PageWithSidebar.Disclosure>
```

Or you can use the `usePage` hook.

```function-live
function Example() {
  const { sidebar } = usePage();
  return (
    <Button onClick={sidebar.toggle}>Toggle sidebar</Button>
  )
}
```

### Minimizing the sidebar

You can minimize the sidebar with your own component using the `<PageWithSidebar.Minimize>` component.

```jsx-live
<PageWithSidebar.Minimize use={Button}>
  Minimize sidebar
</PageWithSidebar.Minimize>
```

Or you can use the `usePage` hook.

```function-live
function Example() {
  const { sidebar } = usePage();
  return (
    <Button onClick={sidebar.toggleMinimize}>Minimize sidebar</Button>
  )
}
```

### Sidebar width

#### Expanded

Customize the width of the expanded sidebar with the `sidebarWidth` prop.

```jsx
<PageWithSidebar
  sidebar={<Box>This is a sidebar</Box>}
  sidebarWidth="250px"
>
  This is the content
</PageWithSidebar>
```

#### Collapsed

Customize the width of the collapsed sidebar drawer (when open in mobile view) with the `collapsedSidebarWidth` prop.

```jsx
<PageWithSidebar
  sidebar={<Box>This is a sidebar</Box>}
  collapsedSidebarWidth="300px"
>
  This is the content
</PageWithSidebar>
```

#### Minimized

Customize the width of the minimized sidebar with the `minimizedSidebarWidth` prop.

```jsx
<PageWithSidebar
  sidebar={<Box>This is a sidebar</Box>}
  minimizedSidebarWidth="30px"
>
  This is the content
</PageWithSidebar>
```
