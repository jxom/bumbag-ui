# Select

`import { Select } from 'fannypack'`

## Basic Usage

The `Select` component renders a `<select>` field with options.

<Playground>
  <Select options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} />
</Playground>

## Default values

For uncontrolled select field, set a default value using the `defaultValue` prop.

<Playground>
  <Select defaultValue="overcast" options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} />
</Playground>

## Placeholders

Add a placeholder to a select field with the `placeholder` prop.

<Playground>
  <Select placeholder="Select weather..." options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} />
</Playground>

## Disabled

Make the select field disabled with the `disabled` prop.

<Playground>
  <Select disabled placeholder="Select weather..." options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} />
</Playground>

## Full width

To make the select field span full width, add the `isFullWidth` prop.

<Playground>
  <Select isFullWidth options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} />
</Playground>

## Sizes

An input can come in different sizes - `small`, `medium`, `large`, or the default.

<Playground>
  <Select size="small" options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} /><br/>
  <Select options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} marginTop="xxsmall" /><br/>
  <Select size="medium" options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} marginTop="xxsmall" /><br/>
  <Select size="large" options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} marginTop="xxsmall" />
</Playground>

## States

An input can use different states (as per palette) such as `danger`, `success` and `warning`.

<Playground>
  <Select state="danger" options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} /><br/>
  <Select state="warning" options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} marginTop="xxsmall" /><br/>
  <Select state="success" options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} marginTop="xxsmall" /><br/>
  <Select state="primary" options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} marginTop="xxsmall" />
</Playground>

## Addons

You can use the [Group component](/components/group) to add on stuff to your `<Select>`.

<Playground>
  <Group>
    <Select options={[{ label: 'Yes', value: 'yes' }, { label: 'No', value: 'no' }]} />
    <Button palette="primary">Submit</Button>
  </Group>
</Playground>

## Controlled Usage

The previous examples are examples of uncontrolled usage. To control the value yourself, use a combination of `onChange` and `value`.

Note: Do not use `defaultValue` with controlled inputs!

<Playground>
  <Component initialState={{ value: 'windy' }}>
    {({ state, setState }) => (
      <Select onChange={value => setState({ value })} value={state.value} options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} />
    )}
  </Component>
</Playground>

## Accessibility

Using an accessibility label (`a11yLabel`), will enable screen readers to read the contents of the label.

<Playground>
  <Select a11yLabel="Weather" options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} />
</Playground>

# SelectField

## Import

`import { SelectField } from 'fannypack'`

## Basic Usage

The `<SelectField>` component contains of the [`<Select>` component](#select) wrapped by the [`<FieldWrapper>` component](/form/fieldwrapper).

It accepts a combination of [`<Select>` props](/form/select#select-props) and [`<FieldWrapper>` props](/form/fieldwrapper#props).

<Playground>
  <SelectField a11yId="weather" label="Weather" options={[{ label: 'Sunny', value: 'sunny' }, { label: 'Windy', value: 'windy' }, { label: 'Overcast', value: 'overcast' }]} />
</Playground>
