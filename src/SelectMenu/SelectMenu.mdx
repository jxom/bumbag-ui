# SelectMenu

## Import

`import { SelectMenu } from 'fannypack'`

## Basic Usage

### Values only

<Playground>
  <SelectMenu options={['Jake Moxey', 'Bill Gates', 'Steve Jobs', 'Mark Zuckerberg']} />
</Playground>

### With label & value

<Playground>
  <SelectMenu
    options={[
      { key: '1', label: 'Jake Moxey', value: { _id: '1', name: 'Jake Moxey' } },
      { key: '2', label: 'Bill Gates', value: { _id: '2', name: 'Bill Gates' } },
      { key: '3', label: 'Steve Jobs', value: { _id: '3', name: 'Steve Jobs' } },
      { key: '4', label: 'Mark Zuckerberg', value: { _id: '4', name: 'Mark Zuckerberg' } }
    ]}
  />
</Playground>

## Default value

There are two ways to define a default value. Either: use the `defaultKey` prop, or add `isDefault` to the option.

<Playground>
  <SelectMenu
    defaultKey="2"
    options={[
      { key: '1', label: 'Jake Moxey', value: { _id: '1', name: 'Jake Moxey' } },
      { key: '2', label: 'Bill Gates', value: { _id: '2', name: 'Bill Gates' } },
      { key: '3', label: 'Steve Jobs', value: { _id: '3', name: 'Steve Jobs' } },
      { key: '4', label: 'Mark Zuckerberg', value: { _id: '4', name: 'Mark Zuckerberg' } }
    ]}
  />
  <SelectMenu
    options={[
      { key: '1', label: 'Jake Moxey', value: { _id: '1', name: 'Jake Moxey' } },
      { key: '2', isDefault: true, label: 'Bill Gates', value: { _id: '2', name: 'Bill Gates' } },
      { key: '3', label: 'Steve Jobs', value: { _id: '3', name: 'Steve Jobs' } },
      { key: '4', label: 'Mark Zuckerberg', value: { _id: '4', name: 'Mark Zuckerberg' } }
    ]}
  />
</Playground>

## Change handler

Use the `onChange` prop to add a change handler. The change handler will provide you with the value and the option as parameters.

<Playground>
  <SelectMenu
    onChange={(value, option) => console.log('Changed!', value, option)}
    options={[
      { key: '1', label: 'Jake Moxey', value: { _id: '1', name: 'Jake Moxey' } },
      { key: '2', label: 'Bill Gates', value: { _id: '2', name: 'Bill Gates' } },
      { key: '3', label: 'Steve Jobs', value: { _id: '3', name: 'Steve Jobs' } },
      { key: '4', label: 'Mark Zuckerberg', value: { _id: '4', name: 'Mark Zuckerberg' } }
    ]}
  />
</Playground>

## Custom option renderer

Use the `optionRenderer` prop if you want to change the content of the menu item.

<Playground>
  <SelectMenu
    optionRenderer={option => (
      <Box>
        <Text fontWeight="600">{option.value._id}.</Text> {option.value.name}
      </Box>
    )}
    options={[
      { key: '1', label: 'Jake Moxey', value: { _id: '1', name: 'Jake Moxey' } },
      { key: '2', label: 'Bill Gates', value: { _id: '2', name: 'Bill Gates' } },
      { key: '3', label: 'Steve Jobs', value: { _id: '3', name: 'Steve Jobs' } },
      { key: '4', label: 'Mark Zuckerberg', value: { _id: '4', name: 'Mark Zuckerberg' } }
    ]}
  />
</Playground>

# SelectMenuField

## Import

`import { SelectMenuField } from 'fannypack'`

## Basic Usage

The `<SelectMenuField>` component contains of the [`<SelectMenu>` component](#selectmenu) wrapped by the [`<FieldWrapper>` component](/form/fieldwrapper).

It accepts a combination of [`<SelectMenu>` props](/form/select#selectmenu-props) and [`<FieldWrapper>` props](/form/fieldwrapper#props).

<Playground>
  <SelectMenuField
    label="Weather"
    options={[
      { key: 1, label: 'Sunny', value: 'sunny' },
      { key: 2, label: 'Windy', value: 'windy' },
      { key: 3, label: 'Overcast', value: 'overcast' }
    ]}
  />
  <SelectMenuField
    isDropdown
    isSearchable
    label="Weather"
    options={[
      { key: 1, label: 'Sunny', value: 'sunny' },
      { key: 2, label: 'Windy', value: 'windy' },
      { key: 3, label: 'Overcast', value: 'overcast' }
    ]}
  />
</Playground>
