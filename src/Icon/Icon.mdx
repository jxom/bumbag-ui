# Icon

Fannypack uses the [free Font Awesome Icon Set](https://fontawesome.com/icons?d=gallery&m=free) for its icons as they provide an extensive and useful set of icons.

**Note:** Fannypack only supports the free Font Awesome icon set right now. Soon you will be able to inject your own font set. Hold tight!

## Import

`import { Icon } from 'fannypack'`

## Basic Usage

<Playground>
  <Icon a11yLabel="Settings" icon="igloo" />
</Playground>

> You can find a list of icon names from the [Font Awesome Icons Directory](https://fontawesome.com/icons?d=gallery&m=free).

## Sizes

Change the size of an icon with the `size` prop.

<Playground>
  <Icon a11yLabel="Settings" icon="igloo" size="small" />
  <Icon a11yLabel="Settings" icon="igloo" marginLeft="xxsmall" />
  <Icon a11yLabel="Settings" icon="igloo" size="medium" marginLeft="xxsmall" />
  <Icon a11yLabel="Settings" icon="igloo" size="large" marginLeft="xxsmall" />
  <Icon a11yLabel="Settings" icon="igloo" size="xlarge" marginLeft="xxsmall" />
  <Icon a11yLabel="Settings" icon="igloo" size="xxlarge" marginLeft="xxsmall" />
  <Icon a11yLabel="Settings" icon="igloo" size="xxxlarge" marginLeft="xxsmall" />
</Playground>

## Colors

Change the color of an icon with the `color` prop. It can be any color from the [palette](/palette), or any other color like `#ff0000` or `red`.

<Playground>
  <Icon a11yLabel="Success" icon="success" color="success" />
  <Icon a11yLabel="Error" icon="danger" color="danger" marginLeft="xxxsmall" />
  <Icon a11yLabel="Warning" icon="warning" color="warning" marginLeft="xxxsmall" />
  <Icon a11yLabel="Copy" icon="clipboard" color="primary" marginLeft="xxxsmall" />
</Playground>

## Usage in text

Icons will be the same size as its text by default.

<Playground>
  <Text>This is text with an icon! <Icon a11yLabel="Thumbs up" icon="thumbs-up" /></Text>
  <Heading use="h3" marginTop="xsmall">This is a heading with an icon! <Icon a11yLabel="Thumbs up" icon="thumbs-up" /></Heading>
</Playground>
