# Primitives

Fannypack exports Reakit's primitives:

- `<Box>`
- `<Block>`
- `<Inline>`
- `<InlineBlock>`
- `<Flex>`
- `<InlineFlex>`
- `<Grid>`

These primitive components provide an abstraction on top of the `<div>` element to include reset styles and allows you to specify CSS attributes as props (see the below for more details).

See more about primitive components [here](https://reakit.io/components/box).

## Box

`import { Box } from 'fannypack'`

All components in Fannypack are built on top of the `<Box>` primitive. Any component in Fannypack can use `Box`'s props. [Read more here](https://reakit.io/components/box).

By default it renders a `<div>` element with basic reset styles.

<Playground>
  <Box>This is a box</Box>
</Playground>

You're able to apply any CSS attribute as a prop to box:

<Playground>
  <Box backgroundColor="primary" color="white" padding="5x">
    This is a box
  </Box>
</Playground>

### Color attributes

Any CSS color attribute (e.g. `color`, `border-color`, etc) can take a [color from the Fannypack palette](/palette#theming).

<Playground>
  <Box backgroundColor="primaryTint" color="primary">
    This is a box
  </Box>
</Playground>

### Margin & padding attributes

Any CSS margin/padding attribute (e.g. `margin`, `padding-left`, etc) can take a [spacing from the Fannypack theme](/styling#layoutthemeconfig).

<Playground>
  <Box backgroundColor="primaryTint" padding="5x">
    This is a box
  </Box>
</Playground>

### Font attributes

The `fontWeight` CSS prop can contain of a [font weight in the Fannypack theme](/styling#fontweightsthemeconfig).

<Playground>
  <Box fontWeight="semibold">This is a box</Box>
</Playground>

The `fontSize` CSS prop can contain of a [font size in the Fannypack theme](/styling#fontsizethemeconfig).

<Playground>
  <Box fontSize="large">This is a box</Box>
</Playground>

## Block

`import { Block } from 'fannypack'`

A `<Block>` is a `<Box>` with `display: block`. [Read more here](https://reakit.io/components/block).

<Playground>
  <Block backgroundColor="primaryTint" padding="5x">
    Block 1
  </Block>
  <Block backgroundColor="successTint" padding="5x">
    Block 2
  </Block>
  <Block backgroundColor="dangerTint" padding="5x">
    Block 3
  </Block>
</Playground>

## Grid

`import { Grid } from 'fannypack'`

A `<Grid>` is a `<Box>` with `display: grid`. It also consists of a `<Grid.Item>` component to pair with the `<Grid>`. [Read more here](https://reakit.io/components/grid).

<Playground>
  <Grid templateColumns="repeat(2, 1fr)" autoRows="auto" gap="3vw">
    <Grid.Item backgroundColor="primaryTint" padding="5x">
      Grid Item 1
    </Grid.Item>
    <Grid.Item backgroundColor="successTint" padding="5x">
      Grid Item 2
    </Grid.Item>
    <Grid.Item backgroundColor="dangerTint" padding="5x">
      Grid Item 3
    </Grid.Item>
  </Grid>
</Playground>
