# Tooltip

## Import

`import { Tooltip } from 'fannypack'`

## Basic Usage

<Playground>
  <Box height="50px">
    <Button>
      Hover on me
      <Tooltip>Hello world!</Tooltip>
    </Button>
  </Box>
</Playground>

## Placement

You can place the tooltip to a particular side of the target with the `placement` prop.

Available values are: `auto`, `top`, `right`, `bottom`, `left`, `top-start`, `right-start`, `bottom-start`, `left-start`, `top-end`, `right-end`, `bottom-end`, `left-end`.

<Playground>
  <Button>
    Hover on me
    <Tooltip placement="right">Hello world!</Tooltip>
  </Button>
</Playground>
