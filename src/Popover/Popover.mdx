# Popover

## Import

`import { Popover } from 'fannypack'`

## Basic Usage

The `<Popover>` component uses `<Hidden>` internally, but it is rendered inside a portal with it's focus trapped (keyboard navigation) for accessibility.

The `content` prop accepts an element, which is the content of your popover. The children of `<Popover>` is the trigger.

<Playground>
  <Box height="100px">
    <Popover content={<Text>Hello world</Text>}>
      <Button>Toggle popover</Button>
    </Popover>
  </Box>
</Playground>

## Placement

To modify the placement of the popover, use the `placement` prop.

<Playground>
  <Popover content={<Text>Hello world</Text>} placement="right">
    <Button>Toggle popover</Button>
  </Popover>
</Playground>

## Close within content

To close from within the content, use the `popover` from `content` with the `<Popover.Hide>` component.

<Playground>
  <Popover
    content={popover => (
      <Box>
        Hello world
        <Button use={Popover.Hide} {...popover}>
          Hide
        </Button>
      </Box>
    )}
    placement="right"
  >
    <Button>Toggle popover</Button>
  </Popover>
</Playground>

Or you can display the close button:

<Playground>
  <Popover content={<Text>Hello world</Text>} placement="right" showCloseButton>
    <Button>Toggle popover</Button>
  </Popover>
</Playground>

## Restricting closure

By default, you can close popovers with the 'esc' key. You can restrict closure via the 'esc' key with the `hideOnEsc` prop.

<Playground>
  <Popover hideOnEsc={false} content={<Text>Hello world</Text>} placement="right">
    <Button>Toggle popover</Button>
  </Popover>
</Playground>

By default, you can also close popovers by clicking outside. You can disable this with the `hideOnClickOutside` prop.

<Playground>
  <Popover hideOnClickOutside={false} content={<Text>Hello world</Text>} placement="right">
    <Button>Toggle popover</Button>
  </Popover>
</Playground>

## Trapping focus

You can trap focus in the popover with the `trapFocus` prop.

Note: you must provide a way to close the popover if you do this.

<Playground>
  <Popover content={<Text>Hello world</Text>} placement="right" showCloseButton trapFocus>
    <Button>Toggle popover</Button>
  </Popover>
</Playground>

# Popover.Popover

## Import

`import { Popover } from 'fannypack'`

## Basic Usage

The `<Popover>` component is made up of a compound of components (such as: `<Popover.Container>`, `<Popover.Show>`, `<Popover.Hide>`, `<Popover.Toggle>`, etc). However, you can use `<Popover.Popover>` to make a more flexible popover.

<Playground>
  <Popover.Container>
    {popover => (
      <InlineBlock relative>
        <Button use={Popover.Toggle} {...popover}>Toggle popover</Button>
        <Popover.Popover use={Pane} border="shadow" padding="small" placement="right" {...popover}>
          <Popover.Close {...popover} />
          Hello world
        </Popover.Popover>
      </InlineBlock>
    )}
  </Popover.Container>
</Playground>
